# 粒子系统说明

## ✨ 功能介绍

已实现从圆心向四周发射粒子的效果！粒子数量根据当前帧的频谱能量（振幅）动态变化。

---

## 🎛️ 配置参数

在 `config.py` 中：

```python
# 粒子系统配置
ENABLE_PARTICLES = True         # 是否启用粒子效果
PARTICLE_SPAWN_RATE = 3.0       # 粒子生成倍率（能量 × 倍率 = 粒子数）
PARTICLE_SPEED = 3.0            # 粒子速度（像素/帧）
PARTICLE_SIZE = 3               # 粒子大小（像素）
PARTICLE_LIFETIME = 30          # 粒子生命周期（帧数）
PARTICLE_FADE = True            # 粒子是否淡出
PARTICLE_COLOR = (255, 255, 255) # 粒子颜色 (RGB)
```

---

## 📊 参数详解

### 1. ENABLE_PARTICLES（开关）

```python
ENABLE_PARTICLES = True   # 启用粒子
ENABLE_PARTICLES = False  # 禁用粒子
```

### 2. PARTICLE_SPAWN_RATE（生成倍率）

**作用**：控制粒子数量

```python
PARTICLE_SPAWN_RATE = 1.0   # 少量粒子
PARTICLE_SPAWN_RATE = 3.0   # 适中（推荐）✅
PARTICLE_SPAWN_RATE = 5.0   # 大量粒子
PARTICLE_SPAWN_RATE = 10.0  # 超多粒子（可能卡顿）
```

**计算公式**：
```python
num_particles = energy × PARTICLE_SPAWN_RATE

例如：
energy = 0.5（平均振幅）
PARTICLE_SPAWN_RATE = 3.0
→ 生成 1-2 个粒子

energy = 0.9（响亮时）
PARTICLE_SPAWN_RATE = 3.0
→ 生成 2-3 个粒子
```

### 3. PARTICLE_SPEED（速度）

**作用**：粒子移动速度

```python
PARTICLE_SPEED = 1.0   # 慢速
PARTICLE_SPEED = 3.0   # 中速（推荐）✅
PARTICLE_SPEED = 5.0   # 快速
PARTICLE_SPEED = 10.0  # 超快
```

**单位**：像素/帧

```
30 fps 视频：
PARTICLE_SPEED = 3.0 → 3像素/帧 = 90像素/秒
```

### 4. PARTICLE_SIZE（大小）

**作用**：粒子圆点半径

```python
PARTICLE_SIZE = 1   # 小点
PARTICLE_SIZE = 2   # 适中
PARTICLE_SIZE = 3   # 正常（推荐）✅
PARTICLE_SIZE = 5   # 大点
PARTICLE_SIZE = 10  # 很大
```

### 5. PARTICLE_LIFETIME（生命周期）

**作用**：粒子存活时间（帧数）

```python
PARTICLE_LIFETIME = 15   # 短命（0.5秒 @ 30fps）
PARTICLE_LIFETIME = 30   # 中等（1秒 @ 30fps）✅
PARTICLE_LIFETIME = 60   # 长命（2秒 @ 30fps）
PARTICLE_LIFETIME = 90   # 很长（3秒 @ 30fps）
```

**效果**：
- 短：粒子快速消失，画面简洁
- 长：粒子留存久，画面饱满

### 6. PARTICLE_FADE（淡出）

**作用**：粒子是否随时间淡化

```python
PARTICLE_FADE = True   # 淡出（推荐）✅
PARTICLE_FADE = False  # 不淡出（突然消失）
```

**效果对比**：
```
PARTICLE_FADE = True:
帧1: ●●●●●● 100% 亮度
帧15: ●●●●●● 50% 亮度
帧30: ●●●●●● 0% 亮度（消失）

PARTICLE_FADE = False:
帧1-29: ●●●●●● 100% 亮度
帧30: 突然消失
```

### 7. PARTICLE_COLOR（颜色）

**作用**：粒子颜色

```python
PARTICLE_COLOR = (255, 255, 255)  # 白色 ✅
PARTICLE_COLOR = (255, 0, 0)      # 红色
PARTICLE_COLOR = (0, 255, 0)      # 绿色
PARTICLE_COLOR = (0, 255, 255)    # 青色
PARTICLE_COLOR = (255, 0, 255)    # 紫色
```

## 🔧 工作原理

### 1. 粒子生成

```python
# 每帧计算能量
energy = np.mean(amplitudes)  # 平均振幅 0-1

# 根据能量决定粒子数量
num_particles = int(energy × 3.0)

# 例如：
能量 0.2 → 0 个粒子（静音）
能量 0.5 → 1-2 个粒子
能量 0.9 → 2-3 个粒子
```

### 2. 粒子运动

```python
# 随机方向（360度）
angle = random.uniform(0, 2π)

# 计算速度向量
vx = PARTICLE_SPEED × cos(angle)
vy = PARTICLE_SPEED × sin(angle)

# 每帧更新位置
x += vx
y += vy
```

### 3. 粒子淡出

```python
# 计算透明度
alpha = 1.0 - (age / lifetime)

# 例如：lifetime=30
age=0:  alpha=1.0 (100%)
age=15: alpha=0.5 (50%)
age=30: alpha=0.0 (消失)

# 应用到颜色
color = (R×alpha, G×alpha, B×alpha)
```

### 4. 粒子清理

```python
# 自动移除死亡粒子
particles = [p for p in particles if p.age < lifetime]
```

## 📈 性能考虑

### 粒子数量 vs 性能

```
PARTICLE_SPAWN_RATE × 平均能量 = 每帧粒子数
每帧粒子数 × PARTICLE_LIFETIME = 屏幕上总粒子数

例如：
PARTICLE_SPAWN_RATE = 3.0
平均能量 = 0.5
PARTICLE_LIFETIME = 30

每帧生成：3.0 × 0.5 = 1-2 个
屏幕总数：2 × 30 = 60 个 ✅ 流畅

如果：
PARTICLE_SPAWN_RATE = 10.0
平均能量 = 0.8
PARTICLE_LIFETIME = 60

每帧生成：10.0 × 0.8 = 8 个
屏幕总数：8 × 60 = 480 个 ⚠️ 可能卡顿
```

### 优化建议

1. **保持粒子总数 < 500**
2. **降低 PARTICLE_SIZE 可以提高性能**
3. **使用 PARTICLE_FADE 减少突然闪烁**

## 🔍 技术实现

### 使用 OpenCV（cv2）绘制

**为什么不用其他库？**
- ✅ `cv2` 已经在项目中使用
- ✅ 无需额外依赖
- ✅ 性能好，绘制简单
- ✅ 与现有代码完美集成

**粒子绘制**：
```python
cv2.circle(frame, (x, y), size, color, -1)
```

### 类结构

```
Particle (单个粒子)
  ├─ 属性：位置(x,y)、速度(vx,vy)、颜色、大小、年龄
  ├─ update(): 更新位置和年龄
  └─ draw(): 绘制到帧上

ParticleSystem (粒子系统)
  ├─ particles: 粒子列表
  ├─ spawn_particles(): 生成新粒子
  ├─ update(): 更新所有粒子
  ├─ draw(): 绘制所有粒子
  └─ get_count(): 获取粒子数量
```
