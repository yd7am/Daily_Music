# 频谱高度控制说明

## 如何限制频谱最高高度

已添加 `MAX_HEIGHT_RATIO` 参数来控制频谱的最大高度。

---

## 🎛️ 在 config.py 中调整

```python
MAX_HEIGHT_RATIO = 0.7  # 最大高度比例
```

### 参数说明

| 值 | 效果 | 视觉高度 |
|----|------|---------|
| **0.3** | 很低 | ███░░░░░░░ 30% |
| **0.5** | 一半 | █████░░░░░ 50% |
| **0.7** | 适中 | ███████░░░ 70% ✅ 当前 |
| **0.8** | 较高 | ████████░░ 80% |
| **1.0** | 全高 | ██████████ 100% |

## 🔧 其他控制高度的方法

### 方法 1：MAX_HEIGHT_RATIO（推荐）✅

```python
# config.py
MAX_HEIGHT_RATIO = 0.7  # 直接控制
```

**优点**：简单直接，一个参数搞定

### 方法 2：调整 DYNAMIC_BOOST

```python
# config.py
DYNAMIC_BOOST = 0.8  # 降低动态增强
```

**效果**：整体降低振幅

### 方法 3：调整频率权重

```python
# config.py
LOW_FREQ_WEIGHT = 0.7   # 降低低频
MID_FREQ_WEIGHT = 0.8   # 降低中频
HIGH_FREQ_WEIGHT = 0.9  # 降低高频
```

**效果**：分频段降低

### 方法 4：修改代码中的 max_height

```python
# audio_visualizer.py 第 259 行
max_height = self.height - 100  # 改为 - 200 降低可用高度
```

**不推荐**：修改代码，不够灵活

## 📝 快速调节步骤

1. **打开 `config.py`**
2. **找到 `MAX_HEIGHT_RATIO`**（第 35 行）
3. **修改数值**：

```python

# 太低了，想增高

# 追求极简
MAX_HEIGHT_RATIO = 0.5

# 追求震撼

```

4. **重新生成视频查看效果**
