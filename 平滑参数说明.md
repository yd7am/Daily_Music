
```python
amplitudes *= 1.0        # 保持当前
smooth_factor = 0.85     # 更平滑
```

### 重低音音乐（EDM、Hip-Hop）
```python
amplitudes *= 0.8        # 再低一点（因为低频很强）
smooth_factor = 0.75     # 适中
```

## 💡 工作原理

### 平滑算法

```python
# 每一帧的计算
当前显示值 = 0.8 × 上一帧的值 + 0.2 × 当前帧的真实值

# 例如：
帧1：真实值 = 100  →  显示 = 100
帧2：真实值 = 50   →  显示 = 0.8×100 + 0.2×50 = 90
帧3：真实值 = 50   →  显示 = 0.8×90 + 0.2×50 = 82
帧4：真实值 = 50   →  显示 = 0.8×82 + 0.2×50 = 76
...慢慢接近 50
```

**优点**：
- 平滑过渡，无突变
- 保持视频帧率（30fps）
- 避免快速闪烁
