# 频率平衡说明

## ❓ 为什么低频振幅大于高频？

这是**音频的真实物理特性**，不是bug！

### 物理原因

```
典型流行音乐的能量分布：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
低频 (20-200Hz)     ████████████ 60-80% ← 贝斯、鼓
中低 (200-500Hz)    ██████░░░░░░ 30-40% ← 男声
中频 (500-2000Hz)   ████░░░░░░░░ 20-30% ← 女声、吉他
中高 (2K-8KHz)      ██░░░░░░░░░░ 10-20% ← 镲片、气音
高频 (8K-20KHz)     █░░░░░░░░░░░ 1-5%   ← 泛音
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**为什么会这样？**

1. **物理特性**
   - 低频波长长（20Hz = 17米），容易产生和传播
   - 低频需要更多能量来推动空气
   - 贝斯和鼓是音乐的"基础"

2. **音乐制作**
   - 混音师会加强低频（让音乐更有力量）
   - 现代流行音乐强调节奏感
   - "低音炮"就是为此设计的

3. **人耳感知**
   - 人耳对 2000-5000Hz 最敏感（听起来很清晰）
   - 但能量确实集中在低频（感觉到震动）

## ✅ 已添加的频率平衡功能

现在可以通过配置文件调整不同频率的显示权重！

### 在 `config.py` 中：

```python
# 频率权重（用于平衡不同频率的显示）
LOW_FREQ_WEIGHT = 0.5       # 低频权重（减半）
MID_FREQ_WEIGHT = 1.0       # 中频权重（保持）
HIGH_FREQ_WEIGHT = 1.5      # 高频权重（增强50%）
```

**效果**：
- 低频显示 ×0.5 = 降低一半
- 中频显示 ×1.0 = 保持不变
- 高频显示 ×1.5 = 增强50%

## 🔧 工作原理

```python
# 代码逻辑
for i in range(num_bars):
    ratio = i / (num_bars - 1)  # 0 到 1
    
    if ratio < 0.3:      # 前30%的条 = 低频
        weight = 0.5
    elif ratio < 0.7:    # 30%-70% = 中频
        weight = 1.0
    else:                # 后30% = 高频
        weight = 1.5
    
    amplitude[i] *= weight
```

## ⚠️ 关于 FREQ_MIN 的建议

您当前设置：
```python
FREQ_MIN = 200  # 从 20 改为 200
```

**影响**：
- ❌ 完全忽略了 20-200 Hz（贝斯和鼓的主频！）
- ✅ 剩下的频率看起来会更均匀

**建议**：
```python
FREQ_MIN = 20   # 改回 20，保留完整低频
# 然后用 LOW_FREQ_WEIGHT 来控制显示强度
LOW_FREQ_WEIGHT = 0.3  # 大幅削弱视觉显示
```

这样既保留了完整的频谱信息，又能控制视觉效果！

## 📝 总结

**为什么低频振幅大？**
- 这是音频的真实物理特性
- 不是程序问题

**如何调整？**
1. **视觉平衡**：调整 `LOW/MID/HIGH_FREQ_WEIGHT`
2. **频率范围**：保持 `FREQ_MIN = 20`，用权重控制
3. **条形数量**：增加 `NUM_BARS` 提高整体分辨率

**推荐配置**：
```python
FREQ_MIN = 20              # 保留完整频谱
NUM_BARS = 100             # 提高分辨率
LOW_FREQ_WEIGHT = 0.5      # 平衡显示
MID_FREQ_WEIGHT = 1.0
HIGH_FREQ_WEIGHT = 1.5
```

这样能得到最好的视觉效果！🎵
