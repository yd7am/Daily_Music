# 输出视频文件目录

## 说明

生成的频谱可视化视频将保存在此目录中。

## 输出格式

- 视频格式：`.mp4`
- 视频编码：H.264 (libx264)
- 音频编码：AAC
- 默认分辨率：1920x1080 (Full HD)
- 默认帧率：30 fps

## 文件命名建议

建议使用描述性的文件名，例如：
- `song_name_spectrum.mp4`
- `song_name_bars_gradient.mp4`
- `song_name_mirror_rainbow.mp4`

这样可以清楚地识别不同的可视化效果版本。

## 文件大小参考

输出视频的文件大小取决于：
- 音频时长
- 视频分辨率
- 视频比特率（默认 5000k）
- 视频复杂度

参考示例：
- 3分钟歌曲（1920x1080, 30fps）：约 100-150 MB
- 5分钟歌曲（1920x1080, 30fps）：约 150-250 MB

## 播放建议

生成的视频可以使用以下播放器播放：
- VLC Media Player
- Windows Media Player
- PotPlayer
- 在线平台（YouTube, Bilibili 等）

## 注意事项

- 确保有足够的磁盘空间（建议至少预留 500MB）
- 首次生成可能需要较长时间（取决于音频时长和电脑配置）
- 可以使用视频编辑软件进一步编辑输出的视频
