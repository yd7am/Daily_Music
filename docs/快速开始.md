# 🚀 快速开始指南

## 第一步：安装 FFmpeg

### Windows 用户（推荐使用 Chocolatey）

1. **安装 Chocolatey**（如果还没有）
   以管理员权限打开 PowerShell，运行：
   ```powershell
   Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
   ```

2. **安装 FFmpeg**
   ```powershell
   choco install ffmpeg
   ```

### 手动安装 FFmpeg（备选方案）

1. 访问 https://ffmpeg.org/download.html
2. 下载 Windows 版本
3. 解压到某个目录（如 `C:\ffmpeg`）
4. 将 `C:\ffmpeg\bin` 添加到系统环境变量 PATH 中

## 第二步：创建虚拟环境

在项目目录中打开 PowerShell：

```powershell
# 进入项目目录
cd D:\MyProject\Daily_Music

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
.\venv\Scripts\Activate.ps1
```

如果遇到权限问题，运行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

---

## 第三步：安装 Python 依赖包

确保虚拟环境已激活（提示符前有 `(venv)`），然后运行：

```powershell
pip install -r requirements.txt
```

这可能需要几分钟时间，请耐心等待。

---

## 第四步：准备音频文件

将您的 MP3 音频文件放入 `input` 目录：

```
Daily_Music/
  └── input/
      └── xxx.mp3  ← 您的音频文件
```

---

## 第五步：生成视频

### 基础用法

```powershell
python main.py --input input/xxx.mp3 --output output/xxx_spectrum.mp4
```

### 高级用法示例

**1. 镜像频谱 + 彩虹色**
```powershell
python main.py -i input/xxx.mp3 -o output/xxx_mirror.mp4 --style mirror --color rainbow
```

**2. 圆形频谱 + 渐变色**
```powershell
python main.py -i input/xxx.mp3 -o output/xxx_circle.mp4 --style circle --color gradient
```

**3. 波形频谱 + 火焰色**
```powershell
python main.py -i input/xxx.mp3 -o output/xxx_wave.mp4 --style wave --color fire
```

**4. 高清 60fps 视频**
```powershell
python main.py -i input/xxx.mp3 -o output/xxx_hd.mp4 --fps 60 --bars 80
```

**5. 添加标题**
```powershell
python main.py -i input/xxx.mp3 -o output/xxx_titled.mp4 --title "我的音乐"
```

## 常见参数说明

| 参数 | 简写 | 说明 | 示例 |
|------|------|------|------|
| `--input` | `-i` | 输入音频文件 | `input/song.mp3` |
| `--output` | `-o` | 输出视频文件 | `output/song.mp4` |
| `--fps` | | 视频帧率 | `30`, `60` |
| `--resolution` | | 视频分辨率 | `1920x1080`, `1280x720` |
| `--bars` | | 频谱条数量 | `60`, `80`, `100` |
| `--color` | | 颜色方案 | `gradient`, `rainbow`, `fire`, `ocean` |
| `--style` | | 显示风格 | `bars`, `mirror`, `circle`, `wave` |
| `--title` | | 标题文字 | `"My Song"` |

## 视觉效果预览

### 风格对比

- **bars** (条形)：经典条形频谱，适合大多数音乐
- **mirror** (镜像)：上下对称效果，视觉冲击力强
- **circle** (圆形)：圆形扩散效果，独特美观
- **wave** (波形)：流动的波浪效果，柔和动感

### 颜色方案对比

- **gradient** (渐变)：青色到品红色，现代科技感
- **rainbow** (彩虹)：七彩渐变，鲜艳活泼
- **fire** (火焰)：红橙黄渐变，热情奔放
- **ocean** (海洋)：蓝绿渐变，清新宁静
- **single** (单色)：纯绿色，简约经典

## 故障排除

### ❌ "找不到 FFmpeg"

**解决方法**：
1. 确认 FFmpeg 已安装：`ffmpeg -version`
2. 如果未安装，参考第一步重新安装
3. 确保 FFmpeg 在系统 PATH 中

### ❌ "modulenotfounderror: no module named 'xxx'"

**解决方法**：
1. 确认虚拟环境已激活
2. 重新安装依赖：`pip install -r requirements.txt`

### ❌ 视频生成很慢

**解决方法**：
1. 降低分辨率：`--resolution 1280x720`
2. 降低帧率：`--fps 24`
3. 减少频谱条数：`--bars 40`

### ❌ 音视频不同步

**解决方法**：
1. 检查音频文件是否完整
2. 尝试转换音频格式为标准 MP3
3. 调整 `config.py` 中的 `HOP_LENGTH` 参数
