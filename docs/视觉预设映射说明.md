# 视觉预设映射说明

## 目标

将历史文档中的调参经验（粒子、动态范围、阈值、平滑）沉淀为可机器读取的预设配置，减少手动反复调参。

## 预设文件

- `web_renderer/src/presets/visual-presets.json`
- `web_renderer/src/presets/music-presets.ts`（读取 JSON 并提供类型）

## 预设列表

### 1. `edm`

- 默认场景：`particleBurst`
- 特征：更低触发阈值 + 更高粒子增益
- 适用：电子、重鼓点、Drop 明显曲目

### 2. `pop`

- 默认场景：`barsNeo`
- 特征：中性阈值 + 中性粒子和色相漂移
- 适用：大多数流行、摇滚、人声主导音乐

### 3. `ambient`

- 默认场景：`circlePulse`
- 特征：更高触发阈值 + 更低粒子增益 + 更慢色相漂移
- 适用：氛围、轻音乐、慢节奏编曲

## 与旧文档关系

以下经验已映射到预设参数中：

- `docs/粒子系统说明.md`：粒子数量和粒子感受强度
- `docs/动态范围增强说明.md`：动态段落强化与平滑
- `docs/自适应能量阈值说明.md`：触发阈值思想

预设并不替代底层分析参数（如 `NOISE_GATE`、`HOP_LENGTH`），而是作为前端视觉编排层的快捷模板。
